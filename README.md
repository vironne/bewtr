# BE WTR — Orchestra Intelligence

Pitch deck statique pour BE WTR par Orchestra Intelligence.

## Stack

Site statique pur — HTML + CSS inline + Google Fonts (Hanken Grotesk, Instrument Serif, JetBrains Mono).

## Déploiement local

```bash
# Aucune build step requise. Ouvrir directement :
start index.html
```

Ou avec un serveur statique :

```bash
npx serve .
```

## Déploiement Vercel

Le site est déployé automatiquement depuis la branche `main`.
